# MindMate Next.js Frontend

This directory is an isolated Next.js refactor track for the current `frontend/` experience.

## Run locally

```bash
npm install
npm run dev
```

## Environment

Use `NEXT_PUBLIC_API_BASE_URL` if you want to point the frontend at a specific deployed Function App.

Example:

```bash
NEXT_PUBLIC_API_BASE_URL=https://func-mindmate-dev-fehce4g0hnfaahgf.canadacentral-01.azurewebsites.net/api
```
